# ✊✋✌️ Rock Paper Scissors Game

A fast and fun web-based **Rock Paper Scissors** game built with HTML, CSS, and JavaScript. <br/> Play against the computer and test your luck in this classic hand game.

## ✨ Features

✅ Play vs Computer: Random computer move generation.  
✅ Visual Feedback: Shows selected choices and results.  
✅ Score Tracking: Tracks wins, losses, and draws.  
✅ Smooth Animations: Engaging UI transitions.  
✅ Fully Responsive: Works on desktop and mobile.

---

## 📂 Files Overview

📜 `index.html` → Layout of game elements and buttons.  
🎨 `style.css` → UI design, layout, and animations.  
🛠️ `script.js` → Game logic, event handling, and result calculation.

---

## 🛠️ Technologies Used

- HTML → Basic structure.  
- CSS → Stylish, responsive design.  
- JavaScript → Game mechanics and UI updates.

---

## 📌 How to Play?

1. Choose **Rock**, **Paper**, or **Scissors**.  
2. The computer will randomly select its move.  
3. The result is displayed along with updated scores.  
4. Keep playing to challenge your luck!
